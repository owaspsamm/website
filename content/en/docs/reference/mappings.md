## Direct mappings

We maintain a [comprehensive mappings spreadsheet](https://docs.google.com/spreadsheets/d/1WiQcn7wFzSM8xg78SqkIM1QF2C48jBCYi_N_6kOq174) covering several standards:

### Microsoft SDL and SAMM
Read more in the [Microsoft SDL and OWASP SAMM Mapping](/blog/comparing-microsoft-sdl-and-samm/) blog post.

### BSIMM 14 and SAMM
Read more in the [BSIMM Mapped to OWASP SAMM](/blog/samm-bsimm-mapping/) blog post.

### IEC-62443-4-1 and SAMM
Available in the same [mappings spreadsheet](https://docs.google.com/spreadsheets/d/1WiQcn7wFzSM8xg78SqkIM1QF2C48jBCYi_N_6kOq174).

### NIST SSDF and SAMM
Created in collaboration with NIST using the [National Online Informative Reference (OLIR) Program](https://csrc.nist.gov/projects/olir). View the [NIST SSDF mapping spreadsheet](https://docs.google.com/spreadsheets/d/1AsIbEHK_csuYkUx8tSZvHBFlywYZ5wBejfJHi8AvnZM). More in the [SAMM-NIST SSDF Mapping](/blog/samm-ssdf-mapping/) blog post.

## OpenCRE

The SAMM model pages link to [OpenCRE](https://www.opencre.org/) in every stream, connecting SAMM to standards like NIST SSDF, ISO27K, PCI-DSS, OWASP ASVS, and NIST 800-53. Learn more in the [SAMM connects to OpenCRE](/blog/owasp-samm-now-connects-to-opencre/) blog post.
