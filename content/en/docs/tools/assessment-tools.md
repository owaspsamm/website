If you want to learn more about SAMM assessments, start with the [assessment page](/assessment/) or the [assessment guide](/docs/assessment/guide/), which includes practical advice, best practices, and [example interview questions](/docs/assessment/example-interview-questions/).

## Spreadsheets

We maintain two versions of the SAMM Toolbox:

- [Microsoft Excel Toolbox](https://github.com/owaspsamm/core/releases/latest)
- [Google Spreadsheet Toolbox](https://docs.google.com/spreadsheets/d/1jmLVltRhuG19AX5cLUcWH1Qox2Uic17rD29gMVG5zDE/view#gid=1716553355)

## Online assessment

[SAMMwise](https://github.com/owaspsamm/sammwise) is our self-hosted online assessment tool. You can find its GitHub repository and documentation there.

[SAMMY](https://sammy.codific.com/) is a third-party tool by Codific, one of our sponsors, that allows you to do online SAMM assessments too.
