[GitHub Discussions](https://github.com/orgs/owaspsamm/discussions/) is a space for longer conversations about the SAMM project: proposals, questions, and feedback that benefit from a threaded format.

## Good use cases

- Proposing changes or additions to the model
- Sharing how you've applied SAMM in your organization
- Providing detailed feedback on documentation or tooling

## Getting started

Head to [SAMM Discussions](https://github.com/orgs/owaspsamm/discussions/) to start a conversation. You'll need a GitHub account to participate.

## For quicker answers

Most day-to-day conversation happens on [Slack](/community/slack/), where core team members and practitioners are active. For bugs or issues with SAMM tools, open an issue directly in the relevant [GitHub repository](https://github.com/owaspsamm).
